## 基于单片机的电梯控制系统设计
## Design of elevator control system based on microcontroller
&emsp;&emsp;刘慧鹏本科重庆邮电大学毕设内容整理，包括程序，PCB，论文。

&emsp;&emsp;如今电梯需求量旺盛，导致电梯的市场快速发展。然而国内电梯市场由外国电梯品牌主导，国内品牌想要提高竞争力，就需要开发自有的电梯控制技术。通过对真实电梯控制系统的调研，根据电梯稳定性、低成本的需求，提出了一种基于单片机的电梯控制系统设计方案。

&emsp;&emsp;将单片机应用在四层电梯控制系统中，完成模型整体结构搭建、电梯主控制板软硬件设计、电脑及手机显示软件设计。电梯采用井道加轿厢的结构，合理安装按钮及传感器，使用井道上方曳引电机控制轿厢上下行。电梯主控制板采用 STM32F103V 型号单片机，处理电梯运行逻辑、外围电路控制、与上位机通信。电梯外围电路设计兼顾可靠性和成本，使用 74HC595 控制数码管显示电路、H 桥搭建电机驱动电路、CH340G 芯片完成串口转 USB 电路、接近开关搭建楼层检测电路。电梯 PC 端上位机程序由可跨平台的图形界面程序 Qt 编写，实现对电梯运行状态、按键状态等的实时状态显示。也可通过手机蓝牙连接主控板，在 app 显示状态。 

&emsp;&emsp;经过对电梯控制系统的测试，电梯控制系统能够完成初始化；按照呼叫信号进行正确的楼层运行；在无信号时待载休眠，有信号时恢复运行。可以看出基于单片机的电梯控制系统实现成本低、运行稳定的需求。
